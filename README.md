# LxvUI

Simplest, but soft UI-kit for you.

## Install

```bash
# Peer dependencies
pnpm add react react-dom tailwindcss@4
# Core package
pnpm add @lanvalird/lxv-ui
```

## Connect to styles

```css
/* tailwindcss.css, style.css or globals.css */
@import "tailwindcss";
@import "@lanvalird/lxv-ui/sources.css";
```
